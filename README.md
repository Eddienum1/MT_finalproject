# MT_finalproject

## Voice Health Journal

This project is a **Voice Health Journal system** that allows users to record daily health information through voice input. The spoken content is converted into text and saved as a log file for later review and analysis.

---

## Project Structure

```text
MT_finalproject/
├─ MT_project.py      # Main program (entry point)
├─ health_log.txt     # Recorded health journal data
├─ README.md          # Project documentation
```

---

## Features

* **Voice Input**: Users can record health-related information using speech
* **Speech-to-Text**: Automatically converts voice input into text
* **Health Log Storage**: Saves records into `health_log.txt`
* **Daily Tracking**: Supports long-term health tracking over time

---

## How to Use

### 1️ Run the Main Program

```bash
python MT_project.py
```

### 2️ Voice Input Workflow

* Follow the on-screen instructions to speak
* The system recognizes and converts speech into text
* The converted content is appended to `health_log.txt`

---

## Notes

* It only recieves Chinese input
* A microphone is required to use this program
* For better recognition accuracy, use the program in a quiet environment
